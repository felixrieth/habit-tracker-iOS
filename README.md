# Habit Tracker iOS

Native SwiftUI version of Habit Tracker Plus.

## Current Scope

This starter includes:

- Native SwiftUI monthly tracking UI
- Daily expandable cards for the eight goals
- Local persistence with `UserDefaults`
- Study hours, protein grams, and bedtime goal logic
- Monthly and yearly summary views

## Open in Xcode

Open `HabitTrackeriOS.xcodeproj` in Xcode and run the `HabitTrackeriOS` scheme on an iPhone simulator or device.

## Goals

- Sports
- Study
- Supplements
- Protein
- Friends
- X
- Read
- Bedtime
